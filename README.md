# ZhiqinZhou.github.io

My personal website built with [Academic Pages](https://github.com/academicpages/academicpages.github.io).

## Quick Start

Edit the following files to customize:

| File | Purpose |
|------|---------|
| `_config.yml` | Site settings, personal info |
| `_pages/about.md` | Homepage content |
| `_data/navigation.yml` | Navigation menu |
| `images/profile.png` | Profile photo |

## Structure

```
├── _config.yml      # Core configuration
├── _pages/          # Page content (about, cv, etc.)
├── _data/           # Data files (navigation, etc.)
├── images/          # Images and favicon
├── files/           # Downloadable files (CV, etc.)
├── _includes/       # Template partials (don't edit)
├── _layouts/        # Page layouts (don't edit)
├── _sass/           # Stylesheets (don't edit)
└── assets/          # JS/CSS assets (don't edit)
```
